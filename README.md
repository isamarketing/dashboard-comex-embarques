# VIVALCE — Dashboard COMEX Embarques

Dashboard interactivo de status y performance de embarques de comercio exterior para VIVALCE S.A.

## URL

https://isamarketing.github.io/dashboard-comex-embarques/

## Vistas

1. **Status Embarques** — KPIs, mapa de milestones visual, detalle por OC con productos
2. **Alertas** — Cambios de ETA, desvíos, configuración de notificaciones
3. **Performance** — Lead time real vs objetivo, descomposición por etapa, por proveedor
4. **Historial** — Todos los embarques 2025-2026
